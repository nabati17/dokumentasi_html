Comment

Untuk menambahkan komentar ke dalam dokumen HTML, Anda dapat menggunakan tag < !-- dan -- > Apa yang ditulis di antara tag < !-- dan -- > akan dianggap sebagai komentar dan tidak akan ditampilkan oleh browser. Ini berguna untuk menambahkan catatan atau instruksi ke dalam kode HTML Anda tanpa mengganggu tampilan atau perilaku halaman web.
