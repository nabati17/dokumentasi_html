Belajar Style

Untuk menambahkan stilistik ke dalam dokumen HTML, Anda dapat menggunakan tag < style > . Tag < style > digunakan untuk menentukan gaya yang akan diterapkan pada elemen HTML tertentu. Gaya ini dapat ditulis dalam bahasa CSS (Cascading Style Sheets) atau dalam bentuk atribut style pada tag HTML.

Contoh style

  • Gunakan background-color untuk warna latar belakang

  • Gunakan color untuk warna teks

  • Gunakan font-family untuk font teks

  • Gunakan font-size untuk ukuran teks

  • Gunakan text-align untuk perataan teks

  • Gunakan font-weight untuk menebalkan teks
