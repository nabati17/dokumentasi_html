Belajar membuat table

Untuk membuat tabel di halaman web menggunakan HTML, Anda dapat menggunakan elemen < table > . Elemen < table > akan menampung seluruh elemen tabel lainnya, seperti baris < tr > dan sel < td >

Contoh Table

nama npm alamat jurusan fakultas
Asnabah 20421075 Teluk Informatika Fakultas Ilmu Komputer
eko 20421076 teluk Pertanian Fakultas Pertanian
Rika Aulia 20421077 pasar ambon Kecantikan Fakultas Kecantian
